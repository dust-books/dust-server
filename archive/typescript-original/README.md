# Original TypeScript Implementation

This directory contains the original TypeScript/Deno implementation of Dust Server,
archived on December 6, 2025 after successful migration to Zig.

## Contents

- `main.ts` - Original server entry point
- `config.ts` - Configuration management
- `database.ts` - Database abstraction layer
- `clock.ts` - Timer/scheduler system
- `module.ts` - Module system interface
- `deno.json` - Deno configuration
- `deno.lock` - Deno dependencies lock file

## Status

This implementation is **no longer active**. The Zig implementation in `src/` is now
the primary codebase.

These files are kept for reference and historical purposes. They can be removed once
the Zig implementation has been fully validated in production.

## Migration Date

December 6, 2025
